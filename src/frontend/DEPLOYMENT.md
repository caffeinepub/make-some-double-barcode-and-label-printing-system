# Deployment Guide

This guide provides step-by-step instructions for deploying and managing the Make Some Double!! label printing application on the Internet Computer.

## Prerequisites

- [dfx](https://internetcomputer.org/docs/current/developer-docs/setup/install/) installed (Internet Computer SDK)
- Node.js and pnpm installed
- Access to the canister deployment environment
- Sufficient cycles for deployment (at least 1T cycles recommended)
